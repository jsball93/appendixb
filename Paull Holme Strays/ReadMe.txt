PHSMesh2D.geo - Example mesh for the Paull Holme Strays 2D inversion, for use in gmsh. The output produces a .msh file, which can be converted to a .dat file for use in R3t.

2D Line 1 - 2D inversion input files for Line 1 at Paull Holme Strays.

2D Line 2 - 2D inversion input files for Line 2 at Paull Holme Strays.

2D Line 3 - 2D inversion input files for Line 3 at Paull Holme Strays.
